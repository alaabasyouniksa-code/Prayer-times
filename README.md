# Welcome App

A simple React Native app initialized with Expo.

## Features

- **Home Screen**: A clean, modern screen with "Welcome" text and a button.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

- Run on Android:
  ```bash
  npm run android
  ```

- Run on iOS (Mac only):
  ```bash
  npm run ios
  ```

- Run on Web:
  ```bash
  npm run web
  ```

## Testing

Run the test suite with:

```bash
npm test
```
